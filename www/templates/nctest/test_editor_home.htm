+++ } } Welcome to the quiz editor - edit questions and parts using the tree on the left.

To add a question, select an existing question and click 'add' -> enter the information into the window and the new question will be added.

To edit a question - double click on its name in the tree or select it and click 'edit' -> edit the information in the window and click save.

To delete a question - select it and click 'delete'.

To add a part - select an existing part and click 'add' - the new part will be added just after.

To edit a part - double click on its name in the tree or select it and click 'edit' -> edit the part in this window.

To delete a part - select it and click 'delete'.

To edit level groups -> double click ' level groups' on the tree, select it and click 'edit' or click the button above.

To edit the quiz instructions s -> double click 'instructions' on the tree, select it and click 'edit' or click the button above.

To view the default instructions that will accompany the quiz -> click the button above.
